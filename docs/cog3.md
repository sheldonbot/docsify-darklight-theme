# Book Cog
This cogs has only one command:
## book
The book command gives you the rating and description of books from goodreads.

### Syntax
If you are using original prefix(sc.):
```
    sc.ascii [book]
```
If you are using your custom prefix:
```
    {prefix}aops [book]
```
